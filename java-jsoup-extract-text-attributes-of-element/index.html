jsoup extract text and attributes of HTML element in Java

Introduction

In this post, we are going to learn how to use the jsoup library in a Java application to extract text and other attributes of an element in the HTML document.

Add jsoup library to your Java project

To use jsoup Java library in the Gradle build project, add the following dependency into the build.gradle file. 

compile 'org.jsoup:jsoup:1.13.1'

To use jsoup Java library in the Maven build project, add the following dependency into the pom.xml file. 

<dependency>
    <groupId>org.jsoup</groupId>
    <artifactId>jsoup</artifactId>
    <version>1.13.1</version>
</dependency>

To download the jsoup-1.13.1.jar file you can visit jsoup download page at jsoup.org/download

Sample HTML File

For example, we have a sample.html file as below. 

<!DOCTYPE html>
<html>
<body>
    <a href="https://simplesolution.dev">Simple <span>Solution</span></a>
</body>
</html>

Extract text and attributes of HTML element

To get text content of an element and all its child elements we can use the Element.text() method and the Element.attr() method can be used to get any attributeâ€™s value of an element. 

import org.jsoup.Jsoup;
import org.jsoup.nodes.Document;
import org.jsoup.nodes.Element;
import org.jsoup.select.Elements;

import java.io.File;
import java.io.IOException;

public class ExtractTextExample {
    public static void main(String... args) {
        try {
            String fileName = "sample.html";
            File file = new File(fileName);
            Document document = Jsoup.parse(file, "UTF-8");
            Elements elements = document.getElementsByTag("a");
            Element element = elements.first();

            String text = element.text();
            String href = element.attr("href");

            System.out.println("Link: " + href);
            System.out.println("Text: " + text);
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}
