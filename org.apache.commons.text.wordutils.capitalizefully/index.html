org.apache.commons.text.WordUtils.capitalizeFully() Java Code Examples

How to use capitalizeFully() method in org.apache.commons.text.WordUtils class.

More Java code examples for org.apache.commons.text.WordUtils Apache Commons Text library.

Java Code Example 1 converts all the whitespace separated words in a String into capitalized words

import org.apache.commons.text.WordUtils;

public class WordUtilsCapitalizeFullyExample1 {
    public static void main(String[] args) {
        String inputString = "how to CAPITALIZE first letter words";

        String outputString = WordUtils.capitalizeFully(inputString);

        System.out.println("Input String: " + inputString);
        System.out.println("Output String: " + outputString);
    }
}
Output:

Input String: how to CAPITALIZE first letter words
Output String: How To Capitalize First Letter Words
