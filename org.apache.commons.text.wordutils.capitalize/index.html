org.apache.commons.text.WordUtils.capitalize() Java Code Examples

How to use capitalize() method in org.apache.commons.text.WordUtils class.

More Java code examples for org.apache.commons.text.WordUtils Apache Commons Text library.

Java Code Example 1 capitalizes all the whitespace separated words in a String

import org.apache.commons.text.WordUtils;

public class WordUtilsCapitalizeExample1 {
    public static void main(String[] args) {
        String inputString = "learn java programming with simple solution website";
        String outputString = WordUtils.capitalize(inputString);

        System.out.println("Input String: " + inputString);
        System.out.println("Output String: " + outputString);
    }
}
Output:

Input String: learn java programming with simple solution website
Output String: Learn Java Programming With Simple Solution Website
