Java Code Example 1 returns null the input String is empty

import com.google.common.base.Strings;

public class StringsEmptyToNullExample1 {
    public static void main(String[] args) {
        String emptyString = "";
        String outputString = Strings.emptyToNull(emptyString);

        System.out.println("Input String: " + emptyString);
        System.out.println("Output String: " + outputString);
    }
}
Output:

Input String: 
Output String: null
